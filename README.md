# 📖 Hello Rails Backend <a name="about-project"></a>

Hello Rails Back-end is a project for creating a Rails API with React Front-end. The goal is to achieve separation of conserns i.e frontend and backend will be to apps that communicate via API.

## 🛠 Built With <a name="built-with"></a>
This app is build with Ruby on Rails and Postgresql database.
### Tech Stack <a name="tech-stack"></a>
> <li><a href="https://www.ruby-lang.org/en/">Ruby</a></li>
> <li><a href="https://rubyonrails.org/">Ruby on Rails</a></li>
> <li><a href="https://www.postgresql.org/">Postgresql</a></li>

### Key Features <a name="key-features"></a>

> - User can create and See a random Greeting message

## 💻 Getting Started <a name="getting-started"></a>

## To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Mac or PC
- Install Ruby
- Install Ruby
- Install Node.js
- Install Yarn
- Install Rails
- Install Postgresql
- Understanding of Ruby

### Frontend

Here is the [Front-End](https://github.com/lengushuru/hello-react-frontend) of this app.

### Setup

- Clone this repository to your desired folder:

```sh
 git clone https://github.com/lengushuru/hello-rails-backend.git
```

- Navigate into the cloned folder

```sh
 cd hello-rails-backend
```
### Install

- Install this project with:

```sh
 bundle install
 yarn install
 npm install
```

- Set up the database
  Change the username and password  in `config/database.yml`

```sh
 rails db:create db:migrate db:seed
```

- Start the server

```sh
 rails s
```
